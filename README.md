# lissome-cli
lissome-cli脚手架工具，主要帮助研发人员迅速生成前端项目模板，减少研发人员手动生成的前端模板的时间以及人工成本，统一管理维护前端脚手架模板，提升研发效率。
目前lissome-cli功能支持如下：
  - 前端基础框架模板，包含react，vue对应的运营端，pc端，移动端3端基础模板
  - 基础模板管理（新增，删除，更新）
  - 版本检测
  - 动态容器化配置模板

# Getting Started
  npm install lissome-cli -g

# commands
  lissome --help
# download template
  lissome init
